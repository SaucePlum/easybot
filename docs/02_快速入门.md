# 快速入门

本指南将帮助你在 5 分钟内完成 EasyBot SDK 的安装、配置并运行第一个 QQ 机器人。

---

## 环境要求

| 项目 | 要求 |
|------|------|
| **Python** | 3.10 或更高版本 |
| **操作系统** | Windows / Linux / macOS |
| **网络** | 可访问 QQ 开放平台 API（`api.sgroup.qq.com`） |
| **QQ 机器人** | 已在 [QQ 开放平台](https://q.qq.com) 创建应用并获取 AppID 和 AppSecret |

### 验证 Python 版本

```bash
python --version
# 输出应 >= Python 3.10.0
```

---

## 安装

### 方式一：从源码安装（推荐开发阶段使用）

```bash
# 克隆或下载项目到本地
cd easybot

# 安装依赖
pip install -r requirements.txt

# 将 easybot 目录加入 Python 路径，或直接在项目根目录运行
```

### 方式二：pip 安装（发布后可用）

```bash
pip install easybot-sdk
```

### 验证安装

```python
import easybot
print(easybot.__version__)  # 应输出: 1.0.0
```

## 获取机器人凭证

在使用 SDK 之前，你需要先在 QQ 开放平台创建机器人应用：

1. 访问 [QQ 开放平台](https://q.qq.com) 并登录
2. 创建一个 **机器人应用**
3. 在应用详情页获取：
   - **AppID**: 应用的唯一标识
   - **AppSecret**: 应用密钥（请妥善保管，勿泄露）
4. 根据需要选择 **公域机器人** 或 **私域机器人**

> **公域 vs 私域**:
> - **公域机器人**: 只能接收频道内 @它的消息（`AT_MESSAGE_CREATE`），可被任意频道添加
> - **私域机器人**: 可以接收频道全量消息、成员变更等更多事件，需手动授权添加到指定频道

## 第一个机器人：Hello World

### 最简示例（~6 行核心代码）

```python
from easybot import Bot

bot = Bot(app_id="你的AppID", app_secret="你的AppSecret")

@bot.on_guild_message
async def on_message(msg):
    await msg.reply("Hello World!")

bot.start()
```

将上述代码保存为 `bot.py`，运行：

```bash
python bot.py
```

在 QQ 频道中 @你的机器人并发送任意消息，它将回复 "Hello World!"。

## 完整示例：多功能 Echo 机器人

以下是一个更完整的示例，展示多种常用功能：

```python
from easybot import (
    Bot,
    MessagesModel,
    Model,
    SandBox,
)

# 初始化机器人
bot = Bot(
    app_id="你的AppID",
    app_secret="你的AppSecret",
    is_private=False,          # 公域机器人
    is_sandbox=True,           # 使用沙箱环境测试
    is_debug=True,             # 开启调试日志
    is_retry=3,                # API 失败重试 3 次
    api_timeout=20,            # API 超时 20 秒
)


# ========== 消息处理 ==========

@bot.on_guild_message
async def handle_guild_message(msg: Model.GuildMessage):
    """处理频道 @机器人消息"""
    content = msg.content.strip()

    if content == "/ping":
        await bot.api.send_guild_message(
            channel_id=msg.channel_id,
            content="Pong! 🏓",
            msg_id=msg.id,
        )

    elif content.startswith("/echo "):
        text = content[6:]  # 去掉 "/echo "
        await msg.reply(text)  # 等价于 send_guild_message + msg_id

    elif content == "/embed":
        embed = MessagesModel.MessageEmbed(
            title="🎉 这是一个 Embed 卡片",
            content=[
                "第一行内容",
                "第二行内容",
                "支持多行文本展示",
            ],
            prompt="点击查看详情",
        )
        await bot.api.send_guild_message(
            channel_id=msg.channel_id,
            content=embed,
        )

    elif content == "/markdown":
        md = MessagesModel.MessageMarkdown(content="# 标题\n\n**粗体文字**\n- 列表项1\n- 列表项2")
        await bot.api.send_guild_message(
            channel_id=msg.channel_id,
            content=md,
        )

    elif content == "/ark":
        ark = MessagesModel.MessageArk23(
            content=["GitHub", "文档", "Issue"],
            link=[
                "https://github.com",
                "https://docs.example.com",
                "https://github.com/issues",
            ],
            desc="快捷链接",
        )
        await bot.api.send_guild_message(
            channel_id=msg.channel_id,
            content=ark,
        )


@bot.on_group_message
async def handle_group_message(msg: Model.GroupMessage):
    """处理群聊 @机器人消息"""
    await bot.api.send_group_message(
        group_openid=msg.group_openid,
        content=f"收到群消息: {msg.content}",
        msg_id=msg.id,
    )


@bot.on_direct_message
async def handle_dm(msg: Model.DirectMessage):
    """处理频道私信"""
    await bot.api.send_direct_message(
        guild_id=msg.guild_id,
        content=f"收到私信: {msg.content}",
        msg_id=msg.id,
    )


# ========== 事件监听 ==========

@bot.on_startup
async def on_startup(event):
    print(f"✅ 机器人已启动! 时间: {event.timestamp}")
    me = await bot.api.get_me()
    print(f"   机器人名称: {me.username} (ID: {me.id})")


@bot.on_shutdown
async def on_shutdown(event):
    print(f"🛑 机器人正在关闭...")


@bot.on_timer(interval=300)
async def heartbeat(event):
    """每 5 分钟执行一次的定时任务"""
    print(f"💓 心跳检查 - 第 {event.tick_count} 次")


if __name__ == "__main__":
    bot.start()
```

## 配置说明

### Bot 构造参数完整列表

```python
bot = Bot(
    # ===== 必填参数 =====
    app_id="your_app_id",              # 机器人 AppID
    app_secret="your_app_secret",      # 机器人 AppSecret

    # ===== 机器人类型 =====
    is_private=False,                  # 是否为私域机器人（默认 False）
    is_sandbox=False,                  # 是否开启沙箱环境（默认 False）

    # ===== 沙箱配置（is_sandbox=True 时生效）=====
    sandbox=SandBox(                   # 沙箱过滤规则
        guilds=["guild_id_1"],         # 白名单频道 ID 列表
        groups=["group_openid_1"],     # 白名单群 openid 列表
    ),

    # ===== 连接协议（默认 WebSocket）=====
    protocol=Proto.websocket(),        # 三选一:
                                       # Proto.websocket()
                                       # Proto.webhook(port=8080)
                                       # Proto.remote_webhook(url="ws://...")

    # ===== API 行为配置 =====
    is_retry=3,                        # API 失败重试次数（默认 3）
    is_log_error=True,                 # 自动记录 API 错误日志（默认 True）
    api_timeout=20,                    # API 请求超时时间/秒（默认 20）

    # ===== 权限与日志 =====
    no_permission_warning=True,        # 权限不足时输出警告（默认 True）
    is_debug=False,                    # 调试模式（默认 False，开启后输出详细日志）

    # ===== 插件系统 =====
    auto_load_plugins=False,           # 自动加载插件目录（默认 False）
    plugins_dir="plugins",             # 插件目录路径（默认 "plugins"）
    plugins_recursive=False,           # 递归扫描子目录（默认 False）
)
```

### 协议选择指南

#### WebSocket 模式（默认）

最常用的方式，SDK 主动连接 QQ 平台的 WebSocket Gateway：

```python
from easybot import Bot, Proto

bot = Bot(
    app_id="xxx",
    app_secret="xxx",
    protocol=Proto.websocket(
        shard_no=0,                           # 分片编号（从0开始）
        total_shard=1,                        # 总分片数
        disable_reconnect_on_not_recv_msg=1000, # 无消息重连间隔/秒
        connect_timeout=30.0,                 # 连接超时/秒
    ),
)
```

**适用场景**: 本地开发、服务器直连部署。

#### Webhook 模式

SDK 启动 HTTP 服务端，QQ 平台主动推送事件：

```python
from easybot import Bot, Proto

bot = Bot(
    app_id="xxx",
    app_secret="xxx",
    protocol=Proto.webhook(
        port=8080,                        # 监听端口
        path="/",                         # Webhook 路径前缀
        path_to_ssl_cert=None,            # SSL 证书路径（反向代理时可省略）
        path_to_ssl_cert_key=None,        # SSL 证书密钥路径
    ),
)
```

**适用场景**: 有公网 IP、使用 nginx 反向代理、云函数部署。

#### Remote Webhook 模式

通过远程中转服务器连接：

```python
from easybot import Bot, Proto

bot = Bot(
    app_id="xxx",
    app_secret="xxx",
    protocol=Proto.remote_webhook(
        url="wss://your-server.com",       # 中转服务器地址
        connect_timeout=15.0,              # 连接超时
        heartbeat_interval=40.0,           # 心跳间隔
        no_msg_timeout=180.0,              # 无消息断线重连
    ),
)
```

**适用场景**: 内网环境、需要通过中转服务器连接的场景。

## 运行与调试

### 启动方式

**同步启动（阻塞主线程）**:

```python
bot.start()  # 阻塞运行，直到 Ctrl+C 停止
```

**异步启动（自行管理事件循环）**:

```python
import asyncio

async def main():
    await bot.start_async()  # 异步启动
    # ... 执行其他逻辑 ...

asyncio.run(main())
```

**上下文管理器（自动清理资源）**:

```python
async with bot:
    await bot.start_async()
    # ... 运行中 ...
# 退出时自动调用 stop_async() 清理资源
```

### 日志查看

SDK 默认将日志输出到两个位置：

1. **控制台**: 彩色格式化输出
2. **文件**: `logs/{bot_id}/{日期}.log`（按天轮转）

开启调试模式以查看更多信息：

```python
bot = Bot(..., is_debug=True)
```

日志格式：`[时间] [级别] [模块] 消息内容`

```
[2026-01-01 12:00:00] [INFO ] [bot   ] 正在启动机器人 (AppID: 123456)
[2026-01-01 12:00:01] [INFO ] [bot   ] 频道@消息事件订阅成功
[2026-01-01 12:00:01] [DEBUG] [http  ] HTTP 客户端已创建
[2026-01-01 12:00:02] [INFO ] [bot   ] 机器人ID: 9876543210，机器人名称: MyBot
```

### 停止机器人

- **Ctrl+C**: 发送中断信号，优雅停止
- **程序调用**: `bot.stop()` 设置停止标志

## 常见消息发送方式速查

```python
# 1. 文本消息
await msg.reply("纯文本")

# 2. 文本 + 图片 URL

# 3. 本地图片
await msg.reply("图片", file_image="./photo.png")

# 4. 引用回复
await msg.reply("回复你", reference=True)

# 5. Embed 卡片
await msg.reply(
    MessagesModel.MessageEmbed(title="标题", content=["内容行"]),
)

# 6. Markdown
await msg.reply(
    MessagesModel.MessageMarkdown(content="# 标题\n**加粗**"),
)

# 7. Ark 模板消息
await msg.reply(
    MessagesModel.MessageArk23(content=["链接"], link=["https://..."]),
)
```
